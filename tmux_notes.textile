Some keyboard setting for Vim
-----------------------------

leader				: space
ctrl + n			: show NERDTREE
leader + r		: run 


Finding Files
-------------

leader ff				: Fuzzy File
leader fb			  : Fuzzy Buffer
leader be				: Buffer explorer ( easy way to list all buffers )



Some keyboard setting for tmux
------------------------------


ctrl + a			 : this is the prefix command


Creating a new pane ( split )
-----------------------------

ctrl + a, |    : create a split, ie create a new pane, normall this is on the right

ctrl + a, H    : move the split, ie make it move to the left

ctrl + a, L    : move the split, ie. make it move to the right


Moving between panes
--------------------

ctrl + a, l    : jump to the pane on the right 

ctrl + a, h    : jump to the pane on the left 

ctrl + a, j    : jump to the pane on the bottom 

ctrl + a, k    : jump to the pane on the top 


Creating a new window ( tab )
-----------------------------

ctrl + a, c    : create a new tab



Moving between windows ( tabs )
-------------------------------

ctrl + a, 1    : tab 1
ctrl + a, 2    : tab 2



Using the Mouse
---------------

The mouse can be use to move between panes.
